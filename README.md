# HESEB

AI 캐릭터들과 SNS·DM을 통해 함께 시간을 보내는 감성 힐링 게임 프로젝트입니다.

## 현재 개발 기준

- 스토리 1~5부 및 외전은 완결 후 시스템에 반영합니다.
- 현재 목표는 인터페이스와 캐릭터 반응 중심의 캐주얼 경험 검증입니다.
- 한 번에 기능 하나만 개발합니다.
- 기능은 테스트와 대표 승인 후에만 `main`에 반영합니다.
- 기존 기능을 깨뜨리지 않고 누적 개발합니다.

## 첫 개발 목표

`V1 STEP 1 — SNS·DM 기본 화면 전환 시스템`

현재 저장소는 기능 개발 전 초기 기준점만 생성된 상태입니다.
